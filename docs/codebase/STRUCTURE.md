## Backend (`server/src/`)

```
server/src/
├── index.ts                # Express app entry: CORS, routes, Socket.io, DB init
├── db/
│   ├── mongodb.ts          # MongoClient singleton, index creation, admin seed
│   └── database.ts         # Legacy SQLite connection (unused in production)
├── middleware/
│   ├── auth.mongo.ts       # authenticateToken, requireAdmin, AuthRequest type
│   └── auth.ts             # Legacy SQLite auth middleware (unused)
├── routes/
│   ├── auth.mongo.ts       # POST /api/auth/login, PUT /api/auth/password
│   ├── conversations.mongo.ts # GET/PATCH /api/conversations (with share/visibility)
│   ├── messages.mongo.ts   # GET /api/messages, POST /api/messages/upload
│   ├── bookmarks.mongo.ts  # GET/POST/DELETE /api/bookmarks
│   ├── search.mongo.ts     # GET /api/search
│   ├── users.mongo.ts      # GET/POST/PUT/DELETE /api/users (admin)
│   ├── rooms.mongo.ts      # GET /api/rooms (HTTP fallback for room listing)
│   └── *.ts                # Legacy SQLite route files (unused)
└── socket/
    ├── auth.ts             # authenticateSocket: JWT validation for Socket.io
    └── handlers.ts         # All socket event handlers (room:*, disconnect)
```

## Entry Points

| Entry               | Path                        | Notes                                     |
|---------------------|-----------------------------|-------------------------------------------|
| Frontend (dev)      | `index.html` → `src/main.tsx` | Vite serves on port 5173                |
| Frontend (prod)     | `dist/index.html`           | Served by nginx container on port 80      |
| Backend             | `server/src/index.ts`       | Compiled to `server/dist/index.js`        |

## Path Aliases

- `@/*` maps to `./src/*` (configured in `tsconfig.app.json` and `vite.config.ts`)
- Never document `dist/`, `server/dist/`, or compiled output

## Non-Source Directories

| Directory         | Purpose                                               |
|-------------------|-------------------------------------------------------|
| `.github/`        | CI/CD workflows + Copilot instruction files           |
| `.playwright-mcp/`| Playwright browser automation session artifacts       |
| `server/data/`    | Legacy SQLite database files (not used in production) |
| `node_modules/`   | Frontend npm packages                                 |
| `server/node_modules/` | Backend npm packages                           |
