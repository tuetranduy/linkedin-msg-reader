## Containerisation

- **Frontend**: multi-stage Docker build → nginx:alpine serving `dist/`
- **Backend**: multi-stage Docker build → node:20-alpine running `dist/index.js`
- Orchestration: Docker Compose (`docker-compose.yml` for dev, `docker-compose.prod.yml` for prod)
- Evidence: `Dockerfile`, `server/Dockerfile`, `docker-compose.prod.yml`

## CI/CD

- **GitHub Actions** — `.github/workflows/deploy.yml`
- Triggers: push to `main`, `workflow_dispatch`
- Deploy job SSHs into VPS, resets to `origin/main`, rebuilds containers
